COVID-19 Chest X-Ray Analysis

Проект анализирует метаданные рентгеновских снимков грудной клетки для мониторинга COVID-19. Используется PySpark для обработки данных, SQL для аналитики и Python (Pandas, Matplotlib, Seaborn) для визуализации.

Используется репозиторий COVID-19 Chest X-Ray Dataset. Основные поля: patientid, age, sex, finding, view, date.

Инструкция по запуску:
Создайте Spark-сессию в Python с названием проекта.
Загрузите CSV-файл метаданных и преобразуйте его в Spark DataFrame.
Выполните предобработку данных: заполнение пропусков, унификация диагнозов, категоризация возраста, удаление дубликатов.
Проведите SQL-запросы для аналитики: статистика по диагнозам, распределение по полу, топ-3 пациента по возрасту, временные тренды, распределение диагнозов по проекциям снимков.
При необходимости сохраните результаты в Parquet.
Для визуализации преобразуйте Spark DataFrame в Pandas DataFrame и создайте графики: круговую диаграмму диагнозов, столбчатую диаграмму возрастных групп, график временных трендов, heatmap распределения диагнозов по проекциям снимков.

Основные визуализации:

Распределение диагнозов среди пациентов.

Распределение пациентов по возрастным категориям.

Временные тренды исследований.

Heatmap диагнозов по типу проекции снимка.


Выводы:
COVID-19 преобладает среди заболеваний грудной клетки.
Пол и возраст влияют на распределение диагнозов.
PySpark эффективно обрабатывает крупные метаданные, а визуализация помогает быстро оценивать ключевые показатели.
